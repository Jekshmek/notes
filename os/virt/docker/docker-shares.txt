In Virtualbox go to settings of your boot2docker machine.
In the last menu add all folders you need.
clb -> D:\clb

And add some code to the
sudo vi /var/lib/boot2docker/bootlocal.sh
mkdir /clb
mount -t vboxsf clb /clb
